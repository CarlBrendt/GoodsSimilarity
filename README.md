Нахождение похожих товаров в интернет-магазинах по их описанию

В output_tf-idf.csv содержится id товара и список id товаров, похожих на него. Для нахождения похожих товаров здесь использовались TF-IDF и косинусовое расстояние(считаем с помощью linear kernel)

В output_word2vec.csv содержится id товара и список id товаров, похожих на него. Для нахождения похожих товаров здесь использовались Word2Vec и косинусовое расстояние(считаем с помощью linear kernel)
